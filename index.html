sbg-synq by tvrhgn

merge data streams for sbggz.nl interface

Dependencies

je hebt een xquery- en xproc-processor nodig (een installatie van calabash bevat beide)

Install

pak sbg-synq uit; zorg ervoor dat het bin/calabash-script werkt. Draai dan de xproc-scripts in src/xproc. Deze scripts verwijzen standaard naar de bestanden in examples. Bestudeer deze om een installatie in eigen omgeving te bouwen.
